# Cyber Ireland 2022 – Agentic Knowledge System

A production-grade agentic backend that converts the Cyber Ireland 2022 PDF into a queryable knowledge system. A LangChain **ReAct agent** orchestrates three specialised tools — semantic search, table queries, and deterministic math — to answer complex questions with verifiable page citations.

---

## Architecture

```
PDF → ETL (pdfplumber + camelot) → Indexing (Chroma + sentence-transformers)
                                         ↓
                       Tools: SemanticSearch | TableQuery | MathTool
                                         ↓
                           ReAct Agent (LangChain + OpenRouter)
                                         ↓
                              FastAPI  POST /query
```

---

## Project Structure

```
pdf-extractor/

├── requirements.txt
├── data/
│   ├── cyber_ireland_2022.pdf   ← place PDF here
│   ├── chunks.json              (generated by ETL)
│   ├── tables/                  (generated by ETL)
│   └── chroma_db/               (generated by Indexing)
├── etl/                 # PDF extraction + cleaning
├── indexing/            # Chunking, embedding, vector store
├── tools/               # SemanticSearch, TableQuery, MathTool
├── agent/               # ReAct agent + JSON logger
├── api/                 # FastAPI server
└── logs/                # Per-request reasoning traces (auto-created)
```

## Setup

### 1. Install dependencies

```bash
pip install -r requirements.txt
```

### 2. Place the PDF

```bash
cp /path/to/cyber_ireland_2022.pdf data/
```

### 3. Run ETL pipeline

```bash
python -m etl.run_etl --pdf data/cyber_ireland_2022.pdf
```

### 4. Build the vector index

```bash
python -m indexing.run_indexing
```

### 5. Start the API

```bash
uvicorn api.main:app --reload --port 8000
```

## Usage

### API Query

```bash
curl -X POST http://localhost:8000/query \
  -H "Content-Type: application/json" \
  -d '{"query": "How many cybersecurity professionals are employed in Ireland?"}'
```

### Response schema

```json
{
  "answer":    "...",
  "citations": [{"page": 12, "text": "..."}],
  "trace":     [{"step": 1, "thought": "...", "action": "SemanticSearch",
                 "action_input": "...", "observation": "..."}]
}
```

## Tools

| Tool | Input | Purpose |
|---|---|---|
| `SemanticSearch` | Natural language query | Retrieve passages with page citations |
| `TableQuery` | JSON `{page, filters, agg}` | Pandas-based structured table queries |
| `MathTool` | JSON `{op, ...}` | CAGR, arithmetic, % change — no LLM |

### MathTool examples

```bash
# CAGR
{"op": "cagr", "start": 5000, "end": 8000, "years": 4}

# Safe eval
{"op": "eval", "expr": "22500 * 1.12 / 100"}

# Percentage change
{"op": "pct_change", "old": 5000, "new": 6500}
```

```json
  "tool_calls": ["SemanticSearch", "MathTool"],
  "steps": [...],
  "answer": "...",

}
```
